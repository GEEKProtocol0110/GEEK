Your Mission: Build the Testnet MVP

Over a 4-6 week period, you will be responsible for building a functional prototype of our Quiz2Earn dApp on the Kaspa Testnet (TN10). This MVP is the most critical asset for our upcoming seed funding proposal.

Core MVP Features to Build:

  • Wallet Connection: A frontend interface that allows users to connect their Kaspa-compatible wallet.

  • Score Tracking: A backend process that validates answers and calculates the score.
  • On-Chain Rewards: An automated transaction that sends a pre-determined amount of Testnet `$GEEK` tokens to the user's wallet upon quiz completion.
